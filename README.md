# Kraken 网站静态复制版

这是 Kraken 加密货币交易所网站的静态复制版，仅用于演示目的。

## 项目说明

- 这是一个纯静态的 HTML/CSS 网站，模仿了 Kraken 加密货币交易所的设计
- 按照要求，所有按钮已被移除，页面不包含任何交互功能
- 网站完全响应式，适配各种屏幕尺寸

## 文件结构

- `index.html` - 主要的 HTML 文件
- `styles.css` - 所有样式定义
- `README.md` - 本文档

## 使用方法

只需在浏览器中打开 `index.html` 文件即可查看网站。
